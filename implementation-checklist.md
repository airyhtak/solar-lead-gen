# Implementation Checklist — Week by Week

## Before Starting
- [ ] Check his employment/contractor agreement — confirm he can run ads
- [ ] Get company permission (or decide personal brand approach)
- [ ] Decide budget: $200/mo (budget) or $500/mo (recommended)

---

## Week 1 — Foundation (Cost: $0)

### Data-Driven Door Knocking (start TODAY)
- [ ] Bookmark Google Project Sunroof: sunroof.withgoogle.com
- [ ] Find NJ county tax records portal for his territory
- [ ] Set up a Google Sheet with scoring columns (address, owner/renter, value, roof direction, score)
- [ ] Score 20 homes before next door-knocking session
- [ ] Plot on Google My Maps for optimized route

### Digital Presence (free)
- [ ] Create/optimize Google Business Profile (category: Solar Energy Contractor or related)
- [ ] Create Nextdoor Business Page (needs company name)
- [ ] Post first 3 Nextdoor posts (education, social proof, Q&A)
- [ ] Join local Facebook groups for his territory (community groups, homeowner groups)

### Lead Infrastructure
- [ ] Create Google Sheet for leads: Date, Name, Email, Phone, Owns Home, Monthly Bill, ZIP, Status, Notes
- [ ] Create free Calendly account for appointment booking
- [ ] Create simple privacy policy page (Google Sites, free)

---

## Week 2 — Paid Traffic + Automation

### Facebook Ads
- [ ] Set up Facebook Business Manager + Ad Account
- [ ] Create 3 ad creatives (savings angle, incentive angle, rate hike angle)
- [ ] Build Lead Form with 6 fields + privacy policy link
- [ ] Launch campaign at $10/day with CBO
- [ ] Set up Zapier: Facebook Lead → Google Sheet + SMS notification to his phone

### Landing Page (if using full stack)
- [ ] Register domain (njsolarsavings.com or similar)
- [ ] Build Carrd.co landing page with calculator
- [ ] Embed Typeform calculator (4 questions)
- [ ] Set up chatbot (Tidio or ManyChat) with qualifying questions
- [ ] Connect chatbot → Calendly → Google Sheet

### SMS Automation
- [ ] Set up GoHighLevel OR Twilio + Zapier for SMS
- [ ] Build 7-message SMS sequence (copy in lead-system-blueprint.md)
- [ ] Include TCPA consent language on all lead capture forms
- [ ] Test entire flow end-to-end: form submit → SMS notification → auto-reply → sequence starts

---

## Week 3 — Content + Optimization

### Content
- [ ] Record first 4 short-form videos (see video ideas in blueprint)
- [ ] Edit with CapCut (free), add captions
- [ ] Post to TikTok, Instagram Reels, YouTube Shorts
- [ ] Batch-generate 12 Nextdoor posts for the month using ChatGPT/Claude

### Facebook Optimization
- [ ] Review ad performance — which angle has lowest CPL?
- [ ] Kill any ad with $50+ spent and zero leads
- [ ] Check lead quality — are they answering the phone? Are they homeowners?

---

## Week 4 — Iterate + Referrals

### Funnel Analysis
- [ ] Calculate: cost per lead, cost per appointment, lead quality score
- [ ] Identify best-performing ad — create 2-3 variations of it
- [ ] Review SMS sequence — are people responding? Where do they drop off?

### Retargeting
- [ ] Create custom audience: lead form openers who didn't submit
- [ ] Launch retargeting ad set ($2/day from main budget)

### Referral System (if he has ANY past installations)
- [ ] Set up referral tracking (unique links in GoHighLevel or manual UTMs)
- [ ] Text past customers the referral offer + their unique link
- [ ] Add referral ask to post-installation process

---

## Ongoing (Monthly)

- [ ] Review + optimize ad spend every Monday (15 min)
- [ ] Generate next month's content batch (scripts via AI, film on phone)
- [ ] Update NJ solar incentive info if rates/programs change
- [ ] Improve SMS sequence based on response data
- [ ] Post 2-3x/week on Nextdoor
- [ ] Score 20 new homes for data-driven door knocking each week
- [ ] Ask every new installation for a Google review
- [ ] Send referral reminders at Day 7, 30, 90 post-install
